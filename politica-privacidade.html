Política de Privacidade

Como tratamos os seus dados no Productive Key

1. Responsável pelo tratamento

Projeto: Productive Key

Responsável: Filipe Veríssimo

Contacto: filipecostaverissimo@gmail.com

2. Dados que podemos recolher

  • Dados de contacto (por exemplo, nome e email) quando enviados por si em formulários de avaliação/feedback.
  • Dados de utilização estritamente necessários para melhorar a experiência da aplicação e do website.
  • Dados de localização aproximada, quando autorizados por si, para apresentar informação meteorológica relevante.

3. Finalidades do tratamento

  • Responder a mensagens e feedback enviados por si.
  • Apresentar funcionalidades dependentes do tempo/localização.
  • Melhorar desempenho, segurança e qualidade da aplicação.

4. Base legal

Tratamos os dados com base no seu consentimento, na execução do serviço solicitado e, quando aplicável, em interesse legítimo para segurança e melhoria contínua.

5. Partilha de dados com terceiros

Podemos recorrer a serviços de terceiros para funcionalidades específicas (por exemplo, envio de emails, chat e dados meteorológicos). Estes prestadores tratam apenas os dados necessários para prestar o serviço.

6. Conservação dos dados

Os dados são conservados apenas pelo período necessário às finalidades indicadas ou conforme obrigação legal. Quando deixarem de ser necessários, serão eliminados de forma segura.

7. Segurança

Adotamos medidas técnicas e organizativas adequadas para proteger os dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição.

8. Direitos do titular dos dados

Nos termos da legislação aplicável (incluindo RGPD), pode solicitar:

  • Acesso aos seus dados;
  • Retificação de dados incorretos;
  • Apagamento dos dados, quando aplicável;
  • Limitação ou oposição ao tratamento;
  • Portabilidade dos dados, quando tecnicamente possível.

Para exercer os seus direitos, contacte-nos através de filipecostaverissimo@gmail.com.

9. Cookies e tecnologias semelhantes

Este website pode utilizar apenas os recursos técnicos necessários ao funcionamento e melhoria da experiência. Se forem introduzidos cookies adicionais, esta política será atualizada.

10. Alterações a esta política

Podemos atualizar esta Política de Privacidade periodicamente. A versão mais recente estará sempre disponível nesta página.
